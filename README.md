# Практическая работа по JavaScript (Задания 1–10)

## Описание
Этот проект содержит 10 простых заданий по JavaScript, которые закрепляют базовые темы:
- переменные и типы данных
- арифметические операции
- условия if/else
- оператор typeof
- switch/case
- циклы for, while, do...while

Все результаты выводятся в консоль (`console.log`).

---

## Содержание заданий

1. Типы данных  
   Созданы переменные: число, строка, логическое значение. Выведены значение и тип через typeof.

2. Арифметика  
   Выполнены: сумма, разность, умножение, деление двух чисел.

3. Проверка возраста  
   Условие: если возраст меньше 18 — доступ запрещён, иначе разрешён.

4. Определение типа значения  
   Проверка типа переменной через typeof и вывод, что это: число, строка, boolean или неизвестный тип.

5. Дни недели (switch)  
   По номеру дня выводится название дня недели.

6. Цикл от 1 до 10  
   Вывод чисел от 1 до 10 через for.

7. Сумма от 1 до 100  
   Посчитана сумма чисел от 1 до 100.

8. Чётные и нечётные (1–20)  
   Для каждого числа выводится: чётное или нечётное.

9. Обратный отсчёт (while)  
   Вывод чисел от 10 до 1.

10. Пароль (do...while)  
   Имитация ввода пароля до тех пор, пока пароль не станет "admin".

## Как запустить

### Вариант 1: через браузер
1. Создай файл index.html
2. Подключи script.js
3. Открой index.html в браузере
4. Открой консоль:
   - Chrome: F12 → вкладка Console
   - Mac: Cmd + Option + J

### Вариант 2: через Node.js
1. Установи Node.js
2. В папке проекта выполни:
   ```bash
   node script.js
